Proyecto Web con Bootstrap

Germán Hincapíe, Programación Web ll, Uniremington

a) ¿Qué es Bootstrap?

Bootstrap es un framework de desarrollo web que permite crear sitios modernos y adaptativos de manera sencilla. Fue creado por Twitter y ofrece una gran variedad de herramientas como estilos CSS predefinidos, componentes visuales interactivos (botones, alertas, menús, etc.), y un sistema de rejilla que facilita el diseño responsivo. Gracias a Bootstrap, es posible construir interfaces visuales limpias, ordenadas y profesionales sin tener que escribir todo el código desde cero. También mejora la compatibilidad entre navegadores y acelera el desarrollo al trabajar sobre una base ya probada y optimizada.

b) ¿Qué se necesita para instalar Bootstrap?

Para instalar Bootstrap se necesita tener conocimientos básicos de HTML y CSS, un editor de código como Visual Studio Code, y un navegador web para visualizar los resultados. Se puede instalar descargando los archivos desde el sitio oficial de Bootstrap o enlazando directamente mediante CDN. También se recomienda tener una carpeta de proyecto organizada para colocar los archivos HTML, CSS y JS que acompañan el desarrollo.

c) ¿Cómo se trabaja Bootstrap a nivel local?

Trabajar con Bootstrap localmente implica descargar los archivos necesarios desde su página oficial (bootstrap.css, bootstrap.js y sus dependencias). Estos archivos se guardan en una carpeta local y se enlazan desde el documento HTML con rutas relativas. Esta forma de trabajo permite tener control total sobre los archivos, modificar estilos según sea necesario y trabajar sin conexión a Internet. Además, es ideal para ambientes cerrados o con restricciones de red.

d) ¿Cómo se configura Bootstrap para trabajar en línea?

Para configurar Bootstrap en línea, se utilizan enlaces CDN (Content Delivery Network), que permiten acceder a los archivos CSS y JS desde servidores externos. Se copian estos enlaces desde el sitio oficial de Bootstrap y se pegan en el encabezado del archivo HTML. Esta opción es ideal para quienes buscan rapidez en la carga del sitio, menor tamaño del proyecto y siempre tener la versión más actualizada del framework sin necesidad de descargar manualmente.
